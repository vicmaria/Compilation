Module Error

module Error: sig .. end

Transforms position informations to string.

val position : Stdlib.Lexing.position -> string

position pos returns a string from a position pos. The string contains the file source and the starting position (line number and character number from the beginning of the line).

val positions : Stdlib.Lexing.position -> Stdlib.Lexing.position -> string

position pos1 pos2 returns a string from two positions pos1 and pos2. The string contains the file source, the starting position and the ending position from that file.
